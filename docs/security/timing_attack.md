# Timing Attack
L'timing attack is une vulnerability qui permet to un attacker de determine la duration d'execution of ae fonction cryptographic. Cela can be used pour determine la secret key or d'autres informations sensibles.

## Threat
L'timing attack can be usede pour determine la secret key of ae fonction cryptographic. Cela can be fait en mesurant la duration d'execution de the function pour differentes inputs.

## Mitigation
Pour prevent les timing attacks, il is recommended d'utiliser of the fonctions cryptographics qui are designeds pour be constant-time. Cela signifie que la duration d'execution de the function ne depends pas de l'input.
