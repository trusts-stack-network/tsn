# Attaque de timing
L'attaque de timing est une vulnérabilité qui permet à un attaquant de déterminer la durée d'exécution d'une fonction cryptographique. Cela peut être utilisé pour déterminer la clé secrète ou d'autres informations sensibles.

## Menace
L'attaque de timing peut être utilisée pour déterminer la clé secrète d'une fonction cryptographique. Cela peut être fait en mesurant la durée d'exécution de la fonction pour différentes entrées.

## Mitigation
Pour prévenir les attaques de timing, il est recommandé d'utiliser des fonctions cryptographiques qui sont conçues pour être constant-time. Cela signifie que la durée d'exécution de la fonction ne dépend pas de l'entrée.
