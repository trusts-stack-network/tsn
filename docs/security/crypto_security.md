# Security of the primitives cryptographics

## Threats

* Timing attacks : les attacks par temps de calcul peuvent permettre to un attacker de recover of the informations sensibles on private keys.
* Side channels : les side channels peuvent permettre to un attacker de recover of the informations sensibles on private keys.
* Mauvais usage of the primitives : un mauvais usage of the primitives cryptographics can permettre to un attacker d'exploiter of the vulnerabilities.

## Mitigations

* Utilisation de primitives cryptographics securees : nous utilisons of the primitives cryptographics tests and securees telles que the signatures ED25519 and les hachages SHA-256.
* Protection contre les timing attacks : nous utilisons of the mechanisms de protection contre les timing attacks tels que les constant-time comparisons.
* Protection contre les side channels : nous utilisons of the mechanisms de protection contre les side channels tels que les masques de security.
