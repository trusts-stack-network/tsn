# TSN UI 2.0 — L'Interface Post-Quantique arrive 🚀

*Annonce officielle — Mars 2026*

## Ce qui arrive

Notre wallet TSN fait peau neuve avec une interface complètement repensée qui reflète enfin notre identité technologique unique. Fini le bleu GitHub générique — place aux **couleurs du code TSN** : cyan laser et violet qubit.

## Pourquoi maintenant ?

Nous construisons la première blockchain post-quantique du monde. Il était temps que notre interface le montre.

### L'ancien design (respectueux mais générique)
- Palette bleue standard du secteur crypto
- Interface fonctionnelle mais sans âme TSN
- Pas de différenciation visuelle vs autres wallets

### Le nouveau design (100% TSN)
- **Cyan laser (#00F5FF)** — la couleur des qubits en action
- **Violet quantique (#7B61FF)** — mystère et innovation cryptographique
- **Glassmorphism** — transparence et modernité
- **Micro-animations** — feedback instantané, sensation premium

## Ce qui change concrètement

### 🎨 Palette Post-Quantique
Les nouvelles couleurs ne sont pas choisies au hasard :
- Le **cyan laser** évoque l'énergie des ordinateurs quantiques
- Le **violet qubit** représente l'innovation cryptographique
- Le **glassmorphism** symbolise la transparence de notre code open source

### ⚡ Effets Dynamiques
- Boutons qui respirent au hover (scaling subtil + glow)
- Soldes qui s'animent lors des mises à jour (compteur fluide)
- Icônes qui pulsent pendant le mining/sync
- Dégradés animés sur les éléments critiques

### 🌓 Thème Adaptatif
Auto-détection clair/sombre basée sur les préférences système — enfin !
