# Philosophie d'animation TSN

## Le manifeste
> "Les transitions quantiques sont instantanées, mais le voyage doit être magnifique."

## Nos principes

### 1. **Fluidité quantique**
Chaque animation reflète la nature ondulatoire de la blockchain — pas de sauts brusques, des courbes naturelles qui respectent la physique (même si c'est la physique quantique).

### 2. **Confiance par la prévisibilité**
Pas de surprises. Les utilisateurs doivent sentir le contrôle même dans les transitions les plus complexes.

### 3. **Performance = priorité**
Une animation qui drop les frames détruit la confiance plus vite qu'un hack. 60fps ou rien.

### 4. **Accessible par design**
`prefers-reduced-motion` n'est pas optionnel — c'est respecter notre communauté.

## Palette d'animation
- **Quantum Blue**: `#3B82F6` - Énergie, confiance, technologie
- **Interstellar Purple**: `#8B5CF6` - Innovation, mystère post-quantique
- **Photon Speed**: `200ms` - L'instant où la transaction devient réelle
- **Neutron Ease**: `ease-out` - Le confort dans l'incertain

## Lexique d'animation
- **Pulse**: Le heartbeat du réseau (hover buttons)
- **Tunnel**: Le voyage entre les états (page transitions)
- **Field**: L'interaction avec la blockchain (parallax)
- **Collapse**: La confirmation finale (wave function collapse)
