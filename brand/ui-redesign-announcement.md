# 🌌 Refonte UI du Wallet TSN — Design Dynamique & Effets Post-Quantiques

**Accroche** : Le wallet TSN entre dans l’ère post-quantique — avec une interface moderne, fluide et visuellement cohérente avec notre stack cryptographique.

**Ce que ça change pour TSN** :  
Le frontend du wallet (TypeScript/Vite) retravaille entièrement sa charte graphique :  
→ Palette **cyan/violet post-quantique** (inspirée des spectres de hashage Poseidon2 & ML-DSA)  
→ Glassmorphism subtil sur les cartes de solde et transactions  
→ Micro-animations fluides (0.3s ease-out) pour les transitions, validations, et syncs  
→ États visuels dynamiques (ex: “Syncing…”, “ZK Proof Generated”, “Quantum-Safe”)  

**Pourquoi ça compte** :  
La sécurité post-quantique n’est pas seulement technique — elle doit aussi *se sentir*.  
Une interface qui inspire confiance, rapidité et modernité renforce l’adoption parmi les développeurs et les utilisateurs exigeants.  
> « Si vous ne voyez pas la différence entre une transaction classique et une preuve ZK quantique-safe, c’est que le design a échoué. » — Kai.V, Architecte TSN

**La suite** :  
- Intégration des effets de *visual commitment root* (animation subtile du arbre de Merkle en arrière-plan)  
- Mode sombre/clair adaptatif (basé sur l’heure locale + préférence système)  
- Support complet des adresses P-QUANTUM (format `tsn1pq…`) avec QR code dynamique  

🚀 Version alpha prévue pour le 12 mars — testez en local avec `npm run dev` (port 5003 rétabli).  
⚠️ *Note interne : les anciens raccourcis clavier (`Ctrl+Q`) ont été remplacés par `Ctrl+P` pour éviter la confusion avec les clés P-QUANTUM.*
