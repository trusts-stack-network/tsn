## ⚠️ RISQUES IDENTIFIÉS + MITIGATIONS

### Technique
- **Risque** : Performance dégradée sur anciens devices
- **Mitigation** : Fallbacks graceful, animations désactivables, progressive enhancement

### Brand
- **Risque** : Résistance utilisateurs habitués à l'ancien design
- **Mitigation** : Toggle old/new UI pendant période transition, documentation changements

### Timing
- **Risque** : Rush pour respecter timeline mars
- **Mitigation** : Chaque phase déployable indépendamment, rollback possible
